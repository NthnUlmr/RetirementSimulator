Detailed Description

A very basic finance simulation which simulates the median person's life until the median retirement age to tell them if they can retire. They can't, but my math could be wrong.

Author
Nathan Ulmer

You can modify the user inputs to see how it compares to your personal retirement plan. the only dependencies are
matplotlib and numpy.
